\pagebreak\section{Comparison}

This section contains comparison of some fonts with their most popular (but
proprietary) competitors.

\subsection{Sans Serif Font Comparison}

The most popular font for sans serif shape is Times New Roman. So let compare it
with our fonts.

\newcommand\fonttimesnewroman{\fontspec{Times New Roman}}

\newcommand\comparefontshelper[4]{
    {#1\testlowerstr\marginpar{#2}}\\
    {#3\testlowerstr\marginpar{#4}}

    {#1\testupperstr\marginpar{#2}}\\
    {#3\testupperstr\marginpar{#4}}
}

\newcommand\comparefonts[4]
{{
    \renewcommand\testbasicstr{\latinpangram}
    \comparefontshelper{#1}{#2}{#3}{#4}

    \renewcommand\testbasicstr{\cyrillicpangram}
    \comparefontshelper{#1}{#2}{#3}{#4}

    \it  % \it command resets default font

    \renewcommand\testbasicstr{\latinpangram}
    \comparefontshelper{#1}{#2}{#3}{#4}

    \renewcommand\testbasicstr{\cyrillicpangram}
    \comparefontshelper{#1}{#2}{#3}{#4}

    \normalfont
    {#1 1234567890\marginpar{#2}}\\
    {#3 1234567890\marginpar{#4}}
}}
